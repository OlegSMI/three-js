
## Начало работы

1. Установка зависимостей
   ```bash
    npm i
    ```
3. Для запуска сервера разработки с hot realod:
    ```bash
    npm run dev
    ```
4. Для сборки исходников:
    ```bash
    npm run build
    ```
5. Для запуска `eslint` на коде проекта:
    ```bash
    npm run lint
    ```
6. Для запуска `eslint` на коде проекта с возможными исправлениями:
    ```bash
    npm run lint:fix
    ```
